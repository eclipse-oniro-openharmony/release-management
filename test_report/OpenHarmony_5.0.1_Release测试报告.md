# **OpenHarmony 5.0.1 Release 测试报告**

## **1** **概述**

 本报告针对OpenHarmony 5.0.1 Release版本的测试验证及质量状态评估，包括对子系统特性和整机专项。

| OpenHarmony 5.0.1.105 版本 | 2024-10-08       | 2024-11-15       |
