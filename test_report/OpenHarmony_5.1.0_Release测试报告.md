# **OpenHarmony 5.1.0 Release 测试报告**

## **1** **概述**

 本报告针对OpenHarmony 5.1.0 Release版本的测试验证及质量状态评估，包括对子系统特性和整机专项。

| OpenHarmony 5.1.0.61 版本 | 2025-4-22        | 2025-4-30        |
